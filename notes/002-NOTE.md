Data structure can be viewed as a systematic way to organize data.

A data type defined a domain of allowed values and the operations that can be performed on those values.

### ADT- Abstract Data Type:
 - define a set of data and a collection of operation can be performed on that data.
 - For definition only mentions what operation are to be performed but not how these operations will be implemented.

#### List ADT
- Definition: 
    - contains elements of the same type arranged in sequential order
- Operations:
  - initilaze, get, insert, remove, removeAt, replace, size, isEmpty, isFull
